Step 1: Stop and Disable Postfix
sudo systemctl stop postfix
sudo systemctl disable postfix



🧼 Step 2: Purge Postfix and Related Packages
This removes Postfix and its configuration files:
sudo apt purge postfix mailutils -y
sudo apt autoremove --purge -y



🧾 Step 3: Clean Up Residual Files
Remove leftover configuration and mail directories:
sudo rm -rf /etc/postfix
sudo rm -rf /var/spool/postfix
sudo rm -rf /etc/mailutils*
sudo rm -rf /home/*/Maildir



🔍 Step 4: Verify Removal
Check if Postfix is still installed or running:
which postfix
postfix check
systemctl status postfix


All should return nothing or indicate it's not found.

🔄 Step 5: Reinstall and Reconfigure Postfix
Now you're ready to start fresh:
sudo apt update
sudo apt install postfix mailutils -y


During installation, you'll be prompted to choose a configuration type. For most setups, select Internet Site and enter your system’s FQDN.

🛠️ Step 6: Reconfigure Postfix (Optional)
If you want to reconfigure without reinstalling:
sudo dpkg-reconfigure postfix
